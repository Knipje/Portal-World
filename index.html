About

Portal World is a streaming tool inspired by Warp World. Like Warp World, the tool allows viewers to submit levels for the streamer to play. The difference is that this application was made for Portal 2 custom levels instead of Super Mario Maker levels. However it does work with all steam workshop items and is thus not limited to only Portal 2 levels, the aesthetics are tooled towards that game though.

This tool is an expansion of an old, extremely simplistic, system I made to display viewer levels quickly on stream, it worked through the streamer manually filling in the level and author name with no added functionality. I kept adding on to that till I had this tool.

Features

  • Level submission through Steam Workshop links
  • Automatic level author detection through Steam
  • A Portal themed bar displaying level information on stream
  • Easily copying level links through the interface

Tutorial

Initial setup

After downloading and installing the program in an easy to reach spot, go to its files and into the "dir" folder. In there open settings.txt

Fill in all fields correctly, keep in mind that the 'NAME' setting refers to the username of the account used to gather the oauth token while 'CHANNEL' refers to the name of the channel the bot will be active on. And that the help command is set to 'commands' by default.

Click here to display image

In this example the bot will be active on Knipje's channel using their own account, allow every user to submit up to 3 levels at any given moment, listen to commands starting with ! and has a help command set to commands (so !commands in chat).

After you have filled in the settings entirely you can launch queuebot.exe (or any shortcuts you may have made during installation) and, if all went well, will be greeted with a menu saying there are no levels in the queue, if you do not get this window please check if you filled everything in correctly or submit a bug report here.

Setting up the stream bar

Setting up the stream bar is relatively easy, you will need to have the program running in order to make it work. Firstly add a new browser source leading to localhost:8000, then fill in the correct width of your stream (most likely 1980) and crop and move the bar as needed. You will know everything is working correctly if the console is getting spammed with messages with code 200, please ignore these as they're mainly used by me for bugfixing.

Chat commands

The proper commands (by default) for the bot are as follows:

Command Variable Aliases Function
!add [Steam Workshop link] !submit Adds a level to the queue
!remove [Level name] !delete Removes a level to the queue
!list None !queue, !q Lists all levels
!current None !currentlevel, !nowplaying, !now, !np Displays the current level
!clear None !reset, !empty Resets the list of levels, mod only command
!commands None None Displays all commands optional

General usage

After the bot has started up normally, viewers will be able to use all commands mentioned above. Remember that the account used for the bot cannot use commands itself. The streamer should then click the "Next Level" button to have the next level appear on screen, click "Copy Link" and paste it into the browser to subscribe to the test chamber, it should be downloaded automatically. To enter a level manually as streamer you can press the corresponding button on the interface, none of the fields are mandatory to be filled in. The queue should persist between uses of the bot, if you want to clear the queue either use the corresponding button or command.

Shutting down the bot

Please remember to shut down the interface first and the console shortly thereafter in order to ensure it closes smoothly.
